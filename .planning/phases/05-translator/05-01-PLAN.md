- `frontend/.env.example` — VITE_MYMEMORY_EMAIL опціональна змінна

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-01 | Tampering | translateApi.ts / fetchJson | mitigate | Перевіряти res.ok і наявність translatedText перед використанням; кидати Error на відсутність даних |
| T-05-02 | Denial of Service | translateApi.ts / REQUEST_TIMEOUT_MS | mitigate | AbortController з 8000ms timeout; apertiumWithFallback не блокує при помилці |
| T-05-03 | Tampering | TranslatorPanel / handleTranslate | mitigate | Byte check new TextEncoder().encode(text).length > 500 до будь-якого fetch |
| T-05-04 | Information Disclosure | VITE_MYMEMORY_EMAIL | accept | Опціональний email у env; якщо не заданий — анонімний режим; прийнятно для персонального застосунку |
| T-05-05 | Spoofing | MyMemory response / translatedText | accept | React автоматично escapes textarea value; XSS неможливий через readOnly textarea |
</threat_model>
