## Component Inventory

### Tab Extension

Extend the existing `tabs tabs-boxed` div in `WordsPage.tsx` (lines 296–313).

```
<button role="tab" className={`tab ${activeTab === 'translate' ? 'tab-active' : ''}`}
  onClick={() => setActiveTab('translate')}>
  Перекладач
</button>
```

`ActiveTab` type: `'analyze' | 'dictionary' | 'translate'`

### Translator Panel Wrapper

```
<main className="pt-6">
  <div className="flex flex-col gap-4 sm:flex-row sm:items-start sm:gap-3">
    {/* source column */}
    {/* swap button column */}
    {/* target column */}
  </div>
  {/* translate button row */}
</main>
```

- Mobile: `flex-col` — textareas stack vertically, swap button between them (centered)
- sm+: `flex-row` — textareas side-by-side, swap button in center column vertically centered

### Language Label

```
<span className="label-text font-semibold text-sm">Українська</span>
<span className="label-text font-semibold text-sm">Nynorsk</span>
```

- Label sits in a `<label className="form-control w-full">` wrapper above each textarea
- When direction is `nn→uk`, swap the label positions accordingly

### Source Textarea

```
<textarea
  className="textarea textarea-bordered w-full resize-none"
  rows={5}
  placeholder="Введи текст для перекладу..."
  value={sourceText}
  onChange={e => setSourceText(e.target.value)}
  disabled={isTranslating}
/>
```

### Result Textarea (read-only)

```
<div className="relative">
  <textarea
    className="textarea textarea-bordered w-full resize-none bg-base-200"
    rows={5}
    readOnly
    value={resultText}
    placeholder="Переклад з'явиться тут"
  />
  {resultText && (
    <button
      type="button"
      className="absolute top-2 right-2 btn btn-ghost btn-xs btn-square"
      onClick={handleCopy}
      aria-label="Скопіювати переклад"
    >
      {/* clipboard icon — 16×16 */}
    </button>
  )}
</div>
```

- Read-only background: `bg-base-200` to distinguish from editable textarea
- Copy button: `btn btn-ghost btn-xs btn-square` — ghost so it does not compete visually with primary CTA
- Copy button only renders when `resultText` is non-empty

### Swap Button

```
<button
  type="button"
  className="btn btn-ghost btn-square self-center mx-auto sm:mt-8"
  onClick={handleSwap}
  aria-label="Поміняти мови місцями"
  disabled={isTranslating}
>
  ⇄
</button>
```

- `self-center` vertically aligns with textarea midpoint on sm+
- `sm:mt-8` offsets for label height above textarea on desktop
- Unicode `⇄` (U+21C4) is the swap glyph — no icon library dependency
- On mobile: rendered between the two textareas in column flow; `mx-auto` centers it

### "Перекласти" Button

```
<Button
  type="button"
  loading={isTranslating}
  disabled={isTranslating || !sourceText.trim()}
  onClick={handleTranslate}
  className="mt-2 sm:mt-0 self-start"
>
  {isTranslating ? 'Перекладаю...' : 'Перекласти'}
</Button>
```

- Uses existing `Button` component from `frontend/src/components/Button.tsx`
- Variant: `primary` (default)
- `loading` prop shows `loading-spinner` (already wired in Button)
- Disabled when input is empty or translation in progress

## States & Interactions

| State | Visual |
|-------|--------|
| Idle (no input) | Source: empty, placeholder visible. Result: empty, placeholder visible. "Перекласти" disabled. |
| Input entered | "Перекласти" enabled. |
| Translating | "Перекласти" shows loading spinner, label "Перекладаю...". Source and swap disabled. |
| Success | Result textarea populated with translated text. Copy button appears. |
| Copy clicked | `toast.success('Скопійовано!')` fires. Toast top-end, 10s, success type. No button state change (stateless). |
| Error | `toast.error('Помилка перекладу. Спробуй ще раз.')` fires. Result textarea cleared or keeps stale text. |
| Swap (text present) | Direction flips. Source text becomes result text and vice versa. Labels swap. Previous result is cleared and source field gets old result value. |
| Swap (empty) | Direction flips. Only labels swap. No text movement. |
