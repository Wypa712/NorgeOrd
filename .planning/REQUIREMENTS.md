# Requirements: Norwegian Learning Hub

**Defined:** 2026-05-27
**Core Value:** Одне поле вводу → AI заповнює все → слово збережено назавжди зі всіма формами і контекстом.

## v1 Requirements

### Authentication

- [ ] **AUTH-01**: Користувач може зареєструватися з email та паролем
- [ ] **AUTH-02**: Користувач може увійти і залишитися в системі після перезавантаження сторінки
- [ ] **AUTH-03**: Користувач може вийти з системи
- [ ] **AUTH-04**: Кожен користувач бачить тільки свої слова

### Words

- [ ] **WORD-01**: Користувач може ввести одне норвезьке слово — AI автоматично заповнює форми (нюнорськ), рід, переклад, приклади речень, теги, рівень складності
- [ ] **WORD-02**: Користувач може переглянути список всіх збережених слів
- [ ] **WORD-03**: Користувач може відкрити детальну картку слова з усією інформацією
- [ ] **WORD-04**: Користувач може відредагувати будь-яке поле збереженого слова
- [ ] **WORD-05**: Користувач може видалити слово зі словника

### Search

- [ ] **SRCH-01**: Користувач може знайти слово за пошуком по норвезькому слову або перекладу

## v2 Requirements

### AI Assistant

- **AIXT-01**: Користувач може задати питання про збережене слово і отримати відповідь від AI
- **AIXT-02**: Контекст розмови зберігається per-word для повторного перегляду

### Tags & Filtering

- **FLTR-01**: Користувач може фільтрувати список за тегами
- **FLTR-02**: Користувач може фільтрувати за рівнем складності (easy/medium/hard)

### Export

- **EXPO-01**: Користувач може експортувати словник у форматі Anki

### Multi-user

- **MULT-01**: Інші користувачі можуть зареєструватися і вести свій словник

## Out of Scope

| Feature | Reason |
|---------|--------|
| SRS / Flashcards | Конкурує з Anki на його полі; краще — Anki export у v2 |
| Bokmål підтримка | Проект для Nynorsk; змішування форм зашкодить навчанню |
| Мобільний додаток | Web-перша версія достатня |
| OAuth (Google/GitHub) | email/password достатньо для v1 |
| Email-верифікація | Особистий інструмент; не критично для v1 |
| Шеринг словників | v2+ після multi-user |
